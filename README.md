# loan_repayment
# 💰 Loan Risk Analysis & Repayment Prediction

## 📌 Overview
This project analyzes loan applicant data to predict whether a customer will repay a loan or default.

## 🚀 Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Decision Tree model for prediction
- Risk analysis insights

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 📊 Insights
- Low income customers show higher default rates
- Loan amount impacts repayment behavior
- Certain patterns help identify risky applicants

## ⭐ Highlight
Built a predictive model to assist financial institutions in identifying high-risk customers.
